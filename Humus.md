# Humus - An Actor-Programming Language

[Humus](http://www.dalnefre.com/wp/humus/) is a pure Actor-based programming language.

```javascript
{
    "lang": "Humus",
    "ast": <block>
}
```

The `ast` is a top-level statement block containing Humus-language statements.
